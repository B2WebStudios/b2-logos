== Description ==

B2 Logos displays logos or other images in a completely customizable carousel slider or grid layout. Developed by [B2 Web Studios](http://b2webstudios.com/ "B2 Web Studios - WordPress Design and Development")

Fully Responsive - B2 Logos adapts to any device screen size, and supports mobile swipe gestures.

Group Logos into Categories - Choose to display only certain logos by selecting one or more categories, such as "Partners", "Clients", or "Sponsors".

Display Logos Anywhere with Shortcodes - Easily generate new shortcodes to display a grid or carousel of logos, loaded with many options for customizing the layout.

Included Widget and Visual Composer Integration - Display any B2 Logos shortcode within a widget or a Visual Composer page layout.

Linking and On Click Event Options - Specify a URL for each logo when clicked, as well as a target (new window or same window) and actions like "Open Link" or "Show Details".

General Settings - Category, number of logos, order (ascending/descending), order by (order, publish date, title, or random).


Fully Customizable Layout - Choose between a slider, grid, or list layout, and customize columns (1 â€“ 10), logo height, spacing, fonts, background colors, and border/divider colors.


Slick Hover Effects - Add hover effects like outer/inner shadows, popup tooltips, borders, fading, or grayscale.

Slider Layout Options - Configure settings for autoplay, transition effects (scroll/fade), easing functions (linear, swing, quadratic, cubic or elastic), and scroll/pause duration.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add new logos by selecting "B2 Logos" -> "Add New Logo" in the WordPress admin menu
4. To create a shortcode, select "B2 Logos" -> "Generate Shortcode" in the WordPress admin menu
